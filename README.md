# ExifViewr

## Demo
https://ryutakojima.github.io/ExifViewer/

## Setting Up Local Environment

1. clone git repo  
  `git clone git@github.com/USERNAME/ExifViewer.git`

2. package install
  `npm install`

## Build

- ESLint  
`npm run eslint`

- SassLint
`npm run sasslint`

- **デバッグ用ビルド**  
`npm run webpack`

- **リリース用ビルド**  
`npm run webpack-production`
